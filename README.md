# Pagina-Objetos-Perdidos
proyecto para la pagina de objetos perdidos hecho como tarea 


Este proyecto es un sitio web destinado a solucionar un problema muy comun en entornos compartidos como lo puede ser una escuela, consiste de dos paginas, la primera es una pagina de login donde los usuarios deben poner sus credenciales para poder acceder al contenido de la pagina principal, en la cual, hay un formulario que se debe completar para publicar un mensaje de objeto perdido u objeto encontrado, por el momento no hay un sistema complejo para login ni un sistema de permisos para distintos usuarios, esto con el objetivo que cualquiera con acceso a este sitio, al ser entendidos como beta-tester con privilegios, deberian poder probar todas las caracteristicas del sitio sin ningun inconveniente.
